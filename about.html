BowTie makes it easy to add user registration, payments, and policy management to otherwise static sites. Our standard templates use Jekyll, but we recently had a user ask us how to populate Angular variables using a BowTie user profile data.

This site, and the accompanying user guide demonstrate a method for injecting BowTie user profile data into Angular. We also walk you through some Angular basics, and point out a few tips for getting Jekyll to work with Angular.

Here is what to expect from this demo:
Account creation is handled by BowTie's hosted user registration page. After a user registers or logs in, they are redirected an 'Edit Profile' page. Here they can fill out or update a few pieces of profile information, save them, and finally get redirected to a page displaying all of their profile information.
