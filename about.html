BowTie makes it easy to add user registration, payments, and policy management to _otherwise_ static sites. Our standard templates use [Jekyll](//jekyllrb.com), but we recently had a user ask us how to leverage the bowtie user profile to populate Angular variables. This site, and the accompanying user guide demonstrate a method for injecting BowTie user profile data into Angular. We also walk you through some Angular basics, and point out a few tips for getting Jekyll to work with Angular. ​
