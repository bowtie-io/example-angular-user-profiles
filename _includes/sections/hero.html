BowTie + Angular

Using BowTie user profiles with Angular

This site demonstrates a method for injecting BowTie user profile data into Angular.

At a recent hackathon, a user asked us how to pass BowTie user profile data into an Angular scope variable. It's not hard to do, but does require an understanding of our templates and how to make Angular play well with Jekyll.
